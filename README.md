Gestion des Réparations d'Appareils Électroniques

Bienvenue sur le dépôt de l'application de gestion des réparations d'appareils électroniques. J'ai développé durant mon stage la partie back-end de cette application, en utilisant PHP Symfony, Doctrine, MySQL.

Pour tester l'application :

Base de Données: Les scripts SQL nécessaires à l'initialisation de la base de données sont disponibles dans le dossier scriptSQL.

Configuration: N'oubliez pas de modifier le fichier .env avec vos propres paramètres de configuration.

Merci pour votre intérêt !
